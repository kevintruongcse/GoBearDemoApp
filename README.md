# GoBearDemoApp

This project is based on the requiment of GoBear company. Built-in by Kotlin

* MinSDK is 21
* Target SDK is 28 

# Feature:
* Intro
* Login
* Parse RSS at http://feeds.bbci.co.uk/news/world/asia/rss.xml
* Show detail article
